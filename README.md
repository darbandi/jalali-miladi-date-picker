# Live Demo
https://jalali-date-picker.netlify.app/


## To get started

1. Clone the repo
```bash
git clone https://github.com/darbandi/jalali-miladi-date-picker.git
```

2. Install dependencies using Yarn
```bash
cd jalali-miladi-date-picker
yarn
```

3. Start the development server
```bash
yarn start
```
